# [AbemaTVAddOn](https://github.com/junyasu0124/AbemaTVAddOn)

Chrome用拡張機能です。AbemaTVに便利な機能を追加します

## 機能
* 再生速度のカスタム
* コメントNGワード（正規表現対応）
* ブラウザ内全画面モード（アーカイブを放送中の番組と同じサイズで視聴できます）
* ユーザーIDの表示

## インストール方法
1. 「[Releases](https://github.com/junyasu0124/AbemaTVAddOn/releases/)」から最新バージョンのファイルをダウンロード
1. ダウンロードしたファイルを展開
1. Chromeで「[chrome://extensions/](chrome://extensions/)」に移動
1. 右上の「デベロッパーモード」を有効にする
1. 「パッケージ化されていない拡張機能を読み込む」をクリック
1. 2で展開されたフォルダーを選択
1. AbemaTVを開いているタブを再読み込み

## 使い方
拡張機能のアイコンをクリックすると表示されるポップアップで、再生速度のカスタム・NGワードの追加をすることができます。またコメントにカーソルを合わせるとユーザーIDが表示されます。
動作しない場合はページを再読み込みしてください。それでも動作しない場合はAbemaTV側の変更が原因と考えられます。拡張機能を一度無効化し、変更に対応したバージョンがリリースされるのをお待ちください。

## 設定の例
- ### NGワード
  - `\p{Extended_Pictographic}` (正規表現有効): 絵文字を含むコメントにマッチ
  - `(.)\1{4}` (正規表現有効): 同じ文字が5文字以上連続するコメントにマッチ
